# 随页

基于 Vue 3、Vite 与 Chrome Manifest V3 的浏览器侧边效率工具箱。网络调试是核心能力之一，同时提供智能填充、开发转换、MFA 动态验证码、二维码、内容对比等可持续扩展的实用工具，并支持浅色、深色和跟随系统主题。

## 当前能力

- 请求 Header 覆写：增加、替换或删除 `X-Forwarded-For`、`X-Real-IP`、`Authorization` 等请求头。
- Query 参数覆写：增加、替换或删除 URL 参数。
- 网络历史：记录请求与响应 Header、Request Body、Response Body、状态和耗时。
- 页面标签分组：按浏览器顶层标签页分组，以当前页面域名和标题识别页面；iframe 请求带独立标识。
- 自动跟随捕获：捕获开启后，切换到其他浏览器标签页或打开新标签页时会自动接入该页面。
- 错误定位：统计 HTTP 4xx/5xx 与网络加载失败；点击错误统计可筛选并查看具体原因。
- 正文体积：按解码后的实际字节数统计，并自动显示为 B、KB 或 MB。
- 请求详情：JSON 语法高亮、代码块单独复制、复制 cURL、复制包含请求与响应的完整快照、导出 HAR。
- 可拉伸详情面板：底部详情卡片支持拖动调整高度，并带进入、退出和页面切换动画。
- 智能填充：覆盖全球国家和地区并生成虚构身份，识别当前网页及 iframe 中的姓名、邮箱、电话、地址和支付字段；支付测试采用 Stripe、Adyen、PayPal 与 Airwallex 官方沙箱场景，扩展不会自动提交表单。
- 可配置导航：顶部功能标签支持横向滚动，并可在设置中按需显示或隐藏。

截图功能当前已按产品规划移除，后续将作为独立模块重新设计。

## 开发与安装

```bash
npm install
npm run build
```

打开 Chrome/Edge 扩展管理页，启用开发者模式，选择“加载已解压的扩展程序”，加载项目中的 `dist` 目录。点击工具栏里的随页图标即可打开侧边栏。

开发预览：

```bash
npm run dev
```

## 权限说明

完整响应正文依赖 `chrome.debugger` / CDP。开始捕获后，Chrome 会显示调试提示，这是浏览器强制的安全提示，扩展无法隐藏。停止捕获会立即断开所有已连接标签页。

MFA 的网页二维码框选和智能填充使用 `chrome.scripting`。前者仅在用户主动点击“框选当前页”后注入临时选择层；后者仅在用户点击“填充当前网页”后识别表单字段并写入当前生成的测试资料。两项操作都不会自动提交网页表单。

请求记录与覆写规则仅保存在本机。请求历史使用 IndexedDB 逐条保存，正文只在打开详情时读取；默认缓存上限为 50 MB，并自动清理超过一小时的历史记录。响应正文、缓存容量和保留时间可在设置中调整。

## 隐私政策

公开隐私政策页面位于 `docs/privacy.html`。提交 Chrome Web Store 前，可通过 GitHub Pages 或任意 HTTPS 静态站点发布该页面，并将公开地址填写到开发者后台的“隐私权政策网址”字段。

Manifest V3 动态规则可以稳定修改请求头与 Query，但修改任意 POST/JSON 请求体需要额外接入 CDP Fetch 拦截；当前版本记录请求体，但暂不在发送前改写。
